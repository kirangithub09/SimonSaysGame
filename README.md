🎮 Simon Says Game

A simple and interactive Simon Says Game built using HTML, CSS, and JavaScript. This project was created to practice JavaScript fundamentals such as DOM manipulation, event handling, arrays, functions, and game logic while recreating the classic memory game.

---

🌟 About the Project

Simon Says is a memory game where the player has to remember and repeat an increasingly long sequence of colored buttons. Every correct sequence takes you to the next level, making the game more challenging as you progress. A single wrong click ends the game, and you can start over with a new sequence.

This project helped me strengthen my understanding of how JavaScript interacts with the DOM and how game logic can be implemented using simple programming concepts.

✨ Features

🎮 Classic Simon Says gameplay
⌨️ Start the game by pressing any key
🎨 Four colored buttons with visual flash effects
📈 Level-based progression
✅ Checks the player's input after every click
❌ Displays the final score when the player makes a mistake
🔄 Easy restart after Game Over

🎯 How to Play

1. Press any key to start the game.
2. Watch carefully as one of the colored buttons flashes.
3. Click the same button.
4. A new color is added to the sequence after each successful level.
5. Repeat the entire sequence correctly to move forward.
6. If you click the wrong button, the game ends and your score is displayed.
7. Press any key to start a new game.
